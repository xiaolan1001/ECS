Detailed Description

Declares the interface for the classes that contains fairness conditions.

Author
Roberto Cavada This interface exports three objects:
  • a generic FairnessList base class
  • a class for justice list (list of BDDs), derived from FairnessList
  • a class for compassion list (couple of BDDs), derived from FairnessList
