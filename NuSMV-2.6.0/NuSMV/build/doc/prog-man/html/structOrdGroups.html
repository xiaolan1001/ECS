Detailed Description

Public interface of class OrdGroups.

Author
Roberto Cavada This class represents a set of groups of variables to be kept grouped

Set of groups of vars.

Variables can be organized in groups, and the class provides methods that allow for searching the group one variable belongs to, and the set of variables that a group contains.

Adds a new variable to the groups set.

The addition is performed only if the variable has not been already added to the same group. If the variable has been already added but to a different group, an error occurs. The group must be already existing.
