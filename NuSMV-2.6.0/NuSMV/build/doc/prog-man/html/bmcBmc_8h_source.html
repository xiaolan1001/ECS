Bmc_GenSolveLtl
int Bmc_GenSolveLtl(NuSMVEnv_ptr env, Prop_ptr ltlprop, const int k, const int relative_loop, const boolean must_inc_length, const boolean must_solve, const Bmc_DumpType dump_type, const char *dump_fname_template)
Given a LTL property generates and solve the problems for all Ki (k_min<=i<=k_max). If bIncreaseK is 0 then k_min==k_max==k and only one problem is generated. If bIncreaseK is 1 then k_min == 0 and k_max == k. Each problem Ki takes into account of all possible loops from k_min to Ki if loopback is &#39;*&#39; (BMC_ALL_LOOPS). Also see the Bmc_GenSolve_Action possible values. Returns 1 if solver could not be created, 0 if everything went smooth.
