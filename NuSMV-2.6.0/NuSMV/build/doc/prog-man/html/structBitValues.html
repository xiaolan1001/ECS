Detailed Description

Public interface of class 'BitValues'.

Author
Roberto Cavada BitValues is a structured array of values of bits. bits are boolean variable that are used to encode a scalar variable. A BitValues is used when extracting scalar value of a variable from the assigments to its bits. A bit value can be BIT_VAL_TRUE, BIT_VAL_FALSE or BIT_VAL_DONTCARE
